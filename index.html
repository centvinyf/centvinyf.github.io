Leetcode-344-Reverse String

Problem Description:

Write a function that takes a string as input and returns the string reversed.

Example:
Given s = “hello”, return “olleh”.

题目大意：

反转一个给定字符串

Code in C++:

class Solution {
public:
    string reverseString(string s) {
        string res="";
        for(int i = 0; i<s.length();i++)
        res=s[i]+res;

        return res;
    }
};

Leetcode刷题记录84-Minimum Path Sum

发表于 2015-10-29   |   分类于 Leetcode   |  

Problem Description:

Given a m x n grid filled with non-negative numbers, find a path from top left to bottom right which minimizes the sum of all numbers along its path.

Note: You can only move either down or right at any point in time.

Solutions:

A simple 2D dynamic programming problem

Code in C++:

class Solution {
    public:
        int minPathSum(vector<vector<int>>& grid) {
            int m=grid[0].size();
            int n=grid.size();
            int solution[n][m];
            solution[0][0]=grid[0][0];
            for(int i=1;i<m;i++)
                solution[0][i]=solution[0][i-1]+grid[0][i];
            for(int j=1;j<n;j++)
                solution[j][0]=solution[j-1][0]+grid[j][0];

            for(int j=1;j<n;j++)
                for(int i=1;i<m;i++)
                    solution[j][i]=min(solution[j-1][i],solution[j][i-1])+grid[j][i];

            return solution[n-1][m-1];

    }
};

Leetcode刷题记录83-Swap Nodes in Pairs

发表于 2015-10-27   |   分类于 Leetcode   |  

Problem Description:

Given a linked list, swap every two adjacent nodes and return its head.

For example,
Given 1->2->3->4, you should return the list as 2->1->4->3.

Your algorithm should use only constant space. You may not modify the values in the list, only nodes itself can be changed.

Solutions:

exchange every pair. remember to keep the pre of next pair.

Code in C++:

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
    public:
        ListNode* swapPairs(ListNode* head) {
            if(!head||!head->next) 
                return head;
            ListNode * a= head;
            ListNode * b= a->next;
            ListNode * newhead =b;
            ListNode* pre= new ListNode(-1);
            pre->next=head;
            while(b&&a)
                {
                    a->next=b->next;
                    b->next=a;
                    pre->next=b;
                    pre=a;
                    a=a->next;
                    if(a)
                    b=a->next;
                    else break;
                }
            return newhead;
       }
};

Leetcode刷题记录82-Sort Colors

发表于 2015-10-27   |   分类于 Leetcode   |  

Problem Description:

Given an array with n objects colored red, white or blue, sort them so that objects of the same color are adjacent, with the colors in the order red, white and blue.

Here, we will use the integers 0, 1, and 2 to represent the color red, white, and blue respectively.

Note:
You are not suppose to use the library’s sort function for this problem.

click to show follow up.

Follow up:
A rather straight forward solution is a two-pass algorithm using counting sort.
First, iterate the array counting number of 0’s, 1’s, and 2’s, then overwrite array with total number of 0’s, then 1’s and followed by 2’s.

Could you come up with an one-pass algorithm using only constant space?

Solution 1:

A typical counting sort problem.

Code in C++ for Solution 1:

class Solution {
    public:
        void sortColors(vector<int>& nums) {
            int numof0=0;
            int numof1=0;
            int numof2=0;
            for(int i=0;i<nums.size();i++)
                {
                    switch(nums[i])
                        {
                            case 0 : numof0++;break;
                            case 1 : numof1++;break;
                            case 2 : numof2++;break;
                            default: break;
                           }
                   }
            for(int i=0;i<numof0;i++)
                nums[i]=0;
            for(int j=0;j<numof1;j++)
                nums[j+numof0]=1;
            for(int k=numof0+numof1;k<nums.size();k++)
                nums[k]=2;

        }
};

Solution 2:

Using two pointers to seprate those parts.

Code in C++ for Solution 2:

class Solution {
    public:
        void sortColors(vector<int>& nums) {
            int indexof0=0;
            int indexof2=nums.size()-1;
            int i=0;
            while(i<=indexof2)
                {
                    if(nums[i]==0)
                        {
                            swap(nums[i],nums[indexof0]);
                            indexof0++;
                            i++;
                        }
                    else if(nums[i]==2)
                        {
                            swap(nums[i],nums[indexof2]);
                            indexof2--;
                        }
                    else i++;
            }

    }
};

Leetcode刷题记录81-Best Time to Buy and Sell Stock

发表于 2015-10-27   |   分类于 Leetcode   |  

Problem Description:

Say you have an array for which the ith element is the price of a given stock on day i.

If you were only permitted to complete at most one transaction (ie, buy one and sell one share of the stock), design an algorithm to find the maximum profit.

Solutions:

This problem is very similar to This problem.Just change the array into the change array.

Code in C++:

class Solution {
    public:
        int maxProfit(vector<int>& prices) {
            if(prices.empty()) 
                return 0;
            int change[prices.size()];
            change[0]=0;
            for(int i=1;i<prices.size();i++)
                change[i]=prices[i]-prices[i-1];
            int res=INT_MIN;
               int sum=0;
            for(int i=0;i<prices.size();i++){
                sum= sum>=0? sum+change[i]:change[i];
                res=max(res,sum);
        }
            return res;
    }
};

Leetcode刷题记录80-Unique Paths

发表于 2015-10-27   |   分类于 Leetcode   |  

Problem Descriptions:

A robot is located at the top-left corner of a m x n grid (marked ‘Start’ in the diagram below).

The robot can only move either down or right at any point in time. The robot is trying to reach the bottom-right corner of the grid (marked ‘Finish’ in the diagram below).

How many possible unique paths are there?

Solutions 1:

Using recursive call. It’s like the Climbing stairs problem.

Code in C++ for Solution 1:

class Solution {
    public:
        int uniquePaths(int m, int n) {
            if(m==1||n==1) return 1;
            else return uniquePaths(m-1,n)+uniquePaths(m,n-1);

    }
};

Solution 2:

Using a map to record the best solutions(using dynamic programming)

Code in C++ for Solution 2:

class Solution {
       public:
           int uniquePaths(int m, int n) {
           int a[m][n];
           for(int i=0;i<m;i++)
                a[i][n-1]=1;
           for(int i=0;i<n;i++)
               a[m-1][i]=1;
           for(int i=m-2;i>=0;i--)
               for(int j=n-2;j>=0;j--)
                   a[i][j]=a[i+1][j]+a[i][j+1];
           return a[0][0];
        }
    };

Leetcode刷题记录79-Find Minimum

发表于 2015-10-27   |   分类于 Leetcode   |  

Problem Description:

Suppose a sorted array is rotated at some pivot unknown to you beforehand.

(i.e., 0 1 2 4 5 6 7 might become 4 5 6 7 0 1 2).

Find the minimum element.

You may assume no duplicate exists in the array.

Solutions:

Using binary search

Code in C++:

class Solution {
    public:
        int findMin(vector<int>& nums) {
            int begin =0;
            int end = nums.size()-1;
            int mid =begin+(end-begin)/2;
            if(nums[end]>nums[begin]) return nums[begin];
            while(end-begin>1)
                {
                    if(nums[mid-1]>nums[mid])
                        return nums[mid];
                    else if(nums[mid]<nums[begin])
                        {
                            end = mid ;
                            mid = begin + (end-begin)/2;
                        }
                    else if (nums[mid]>nums[begin])
                        {
                            begin = mid;
                            mid = begin + (end-begin)/2;
                        }
                }
            return min(nums[end],nums[begin]);

        }
};

Leetcode刷题记录78-Convert Sorted Array to Binary Search Tree

发表于 2015-10-25   |   分类于 Leetcode   |  

Problem Description:

Given an array where elements are sorted in ascending order, convert it to a height balanced BST.

Solutions:

Do it recursively. use mid element to be the root.

Code in C++:

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
    public:
        TreeNode* sortedArrayToBST(vector<int>& nums) {
            vector<int> a;
            vector<int> b;
            int mid=nums.size()/2;
            for(int i=0;i<mid;i++)
            a.push_back(nums[i]);
            for(int i=mid+1;i<nums.size();i++)
            b.push_back(nums[i]);
            if(nums.size()==0) 
                return NULL;
            else if(nums.size()==1)
                {
                    TreeNode* t= new TreeNode(nums[0]);
                    return t;
                }
            else
                {
                       TreeNode* root =new TreeNode(nums[mid]);
                       root->left= sortedArrayToBST(a);
                       root->right=sortedArrayToBST(b);
                       return root;
                }
        }
    };

Leetcode刷题记录77-Maximum Subarray

发表于 2015-10-25   |   分类于 Leetcode   |  

Problem Descriptions:

Find the contiguous subarray within an array (containing at least one number) which has the largest sum.

For example, given the array [−2,1,−3,4,−1,2,1,−5,4],
the contiguous subarray [4,−1,2,1] has the largest sum = 6.

Solutions:

Once the sum is less than 0, start from the current element.

Code in C++:

class Solution {
    public:
        int maxSubArray(vector<int>& nums) {
            int res=INT_MIN;
            int sum=0;
            for(int i=0;i<nums.size();i++){
                sum= sum>=0? sum+nums[i]:nums[i];
                res=max(res,sum);
            }
        return res;
    }
};

Leetcode刷题记录76-Integer to Roman

发表于 2015-10-25   |   分类于 Leetcode   |  

Problem Description:

Given an integer, convert it to a roman numeral.

Input is guaranteed to be within the range from 1 to 3999.

Solutions:

Like coin pick problems, just use greedy algorithm

Code in C++:

class Solution {
    public:
        string intToRoman(int num) {
            string res="";
            int a[13]={1000,900,500,400,100,90,50,40,10,9,5,4,1};
            string x[13]={"M","CM","D","CD","C","XC","L","XL","X","IX","V","IV","I"};
            for(int i=0;i<13;i++)
                {
                    if(num>=a[i])
                        {
                            for(int j=0;j<num/a[i];j++)
                                res+=x[i];
                                num%=a[i];
                        }
                }
            return res;

        }
    };
