Leetcode刷题记录75-Search Insert Position

发表于 2015-10-25   |   分类于 Leetcode   |  

Problem Description:

Given a sorted array and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

You may assume no duplicates in the array.

Here are few examples.
[1,3,5,6], 5 → 2
[1,3,5,6], 2 → 1
[1,3,5,6], 7 → 4
[1,3,5,6], 0 → 0

Solutions:

Using binary search

Code in C++:

class Solution {
    public:
        int searchInsert(vector<int>& nums, int target) {
            int begin =0;
            int end = nums.size()-1;
            int mid = begin+(end-begin)/2;
            while(end-begin>1)
                    {
                    if(nums[mid]==target) 
                        return mid;
                    else if(nums[mid]>target)
                        {
                            end=mid;
                            mid=begin+(end-begin)/2;
                            continue;
                        }
                    else if(nums[mid]<target)
                        {
                            begin=mid;
                            mid=begin+(end-begin)/2;
                            continue;
                        }
                }
            if(target<=nums[begin])return begin;
            if(target>nums[end]) return end+1;
            return end;

    }
};

Leetcode刷题记录74-Unique Binary Search Tree

发表于 2015-10-25   |   分类于 Leetcode   |  

Problem Description:

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?

For example,
Given n = 3, there are a total of 5 unique BST’s.

1 3 3 2 1
\ / / / \ \
3 2 1 1 3 2
/ / \ \
2 1 2 3

Solutions:

A unique binary search tree using i as root in n nodes are (0~i-1) in the left and (i+1~n) in the right. So we can do a dynamic programming.

Updates:
You can see from here . This kind of number is called “Catalan number”.

Code in C++:

class Solution {
    public:
        int numTrees(int n) {
            int res[n+1]={0};
            res[0]=1;
            res[1]=1;
            for(int i=2;i<=n;i++)
                    for(int j=0;j<i;j++)
                      res[i]+=res[j]*res[i-1-j];
            return res[n];
    }
};

Leetcode刷题记录73-Missing Number

发表于 2015-10-24   |   分类于 Leetcode   |  

Problem Description:

Given an array containing n distinct numbers taken from 0, 1, 2, …, n, find the one that is missing from the array.

For example,
Given nums = [0, 1, 3] return 2.

Note:
Your algorithm should run in linear runtime complexity. Could you implement it using only constant extra space complexity?

Solution 1:

If we have O(n) extra space. we can mark every appeard element.

Code in C++ for Solution 1:

class Solution {
    public:
        int missingNumber(vector<int>& nums) {
            bool flag[nums.size()+1];
            for(int i=0;i<nums.size();i++) 
                flag[i]=false;
            for(int i=0;i<nums.size();i++)
                flag[nums[i]]= true;
            for(int i=0;i<nums.size();i++)
                if(!flag[i]) return i;
            return nums.size();

    }
};

Solution 2:

We can add the total number and minus every element.

Code in C++ for Solution 2:

class Solution {
    public:
        int missingNumber(vector<int>& nums) {
            int res=nums.size();
            for(int i=0;i<nums.size();i++)
                res += (i-nums[i]);
            return res;

    }
};

Solution 3:

Use Bit Manipulation(which I don’t know very well)

Code in C++ for Solution 3:

class Solution {
    public:
        int missingNumber(vector<int>& nums) {
            int res = 0;
            for (int i = 0; i < nums.size(); ++i) {
                res ^= (i + 1) ^ nums[i];
        }
            return res;
    }
};

Leetcode刷题记录72-Populating Next Right Pointers in Each Node

发表于 2015-10-24   |   分类于 Leetcode   |  

Problem Description:

Given a binary tree

struct TreeLinkNode {
  TreeLinkNode *left;
  TreeLinkNode *right;
  TreeLinkNode *next;
}

Populate each next pointer to point to its next right node. If there is no next right node, the next pointer should be set to NULL.

Initially, all next pointers are set to NULL.

Note:

You may only use constant extra space.
You may assume that it is a perfect binary tree (ie, all leaves are at the same level, and every parent has two children).
For example,
Given the following perfect binary tree,
1
/ \
2 3
/ \ / \
4 5 6 7
After calling your function, the tree should look like:
1 -> NULL
/ \
2 -> 3 -> NULL
/ \ / \
4->5->6->7 -> NULL

Solutions:

It’s very like This Problem

Code in C++:

 /**
 * Definition for binary tree with next pointer.
 * struct TreeLinkNode {
 *  int val;
 *  TreeLinkNode *left, *right, *next;
 *  TreeLinkNode(int x) : val(x), left(NULL), right(NULL), next(NULL) {}
 * };
 */
class Solution {
    public:
        void connect(TreeLinkNode *root) {
            queue<TreeLinkNode*> a;
            if(root)
                {
                    TreeLinkNode* pre=NULL;
                    a.push(root);
                    a.push(NULL);
                    while(!a.empty())
                        {
                            TreeLinkNode* t=a.front();
                            a.pop();
                            if(t)
                                {
                                    if(t->left)
                                        a.push(t->left);
                                    if(t->right)
                                        a.push(t->right);
                                    if(pre) 
                                        pre->next = t;
                                    pre=t;                                 
                                }
                            else{
                                    if(!a.empty())
                                        {
                                            a.push(NULL);
                                            if(pre) 
                                                pre->next=NULL;
                                            pre=NULL;
                                        }
                                }

                        }
                }

        }
    };

Leetcode刷题记录71-Linked List Cycle

发表于 2015-10-24   |   分类于 Leetcode   |  

Problem Description:

Given a linked list, determine if it has a cycle in it.

Follow up:
Can you solve it without using extra space?

Solution 1:

If we can have extra space

Code in C++ for Solution 1:

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
    public:
        bool hasCycle(ListNode *head) {
            set<ListNode*> s;
            ListNode*a = head;
            while(a)
                {
                    if(s.find(a)!=s.end()) 
                        return true;
                    else 
                        {
                               s.insert(a);
                               a=a->next;
                        }

                }
            return false;

        }        
    };

Solution 2:

If we don’t have extra space. We can have two pointers, once there being a cycle, they will meet.

Code in C++ for Solution 2:

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
    public:
        bool hasCycle(ListNode *head) {
            if(!head) 
                return false;
            ListNode *a = head;
            ListNode *b = NULL;
            if(a->next)
                b=a->next;
            while(a&&b)
                {
                       if(a==b) 
                           return true;
                       else
                           {
                               if(b->next) 
                                   b=b->next->next;
                               else 
                                   return false;
                               a=a->next;
                           }

                }
            return false;

        }
    };

Leetcode刷题记录70-Binary Search Tree InOrder Traversal

发表于 2015-10-24   |   分类于 Leetcode   |  

Problem Description:

Given a binary tree, return the inorder traversal of its nodes’ values.

For example:
Given binary tree {1,#,2,3},
1
\
2
/
3
return [1,3,2].

Note: Recursive solution is trivial, could you do it iteratively?

Solutions:

It’s like This Problem

Code in C++:

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
    public:
        vector<int> inorderTraversal(TreeNode* root) {
                vector<int> res;
            stack<TreeNode*> s;
            TreeNode*p = root;
            while(p||!s.empty())
                {
                    while(p)
                        {
                            s.push(p);
                            p=p->left;
                        }
                    if(!s.empty())
                        {
                               res.push_back(s.top()->val);
                               p=s.top()->right;
                               s.pop();
                        }
                }
            return res;

        }
    };

Leetcode刷题记录69-Binary Tree Preorder Traversal

发表于 2015-10-24   |   分类于 Leetcode   |  

Problem Description:

Given a binary tree, return the preorder traversal of its nodes’ values.

For example:
Given binary tree {1,#,2,3},
1
\
2
/
3
return [1,2,3].

Note: Recursive solution is trivial, could you do it iteratively?

Solutions:

Use a stack to store the pre nodes.

Code in C++:

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
    public:
        vector<int> preorderTraversal(TreeNode* root) {
            vector<int> res;
            TreeNode* p = root;
            stack<TreeNode*> s;
            while(!s.empty()||p)
                {
                    while(p)
                        {
                            s.push(p);
                            res.push_back(p->val);
                            p=p->left;
                        }
                    if(!s.empty())
                        {
                            p=s.top()->right;
                            s.pop();
                        }
                }
            return res;

            }
    };

Leetcode刷题记录68-Product Of Array Except Self

发表于 2015-10-15   |   分类于 Leetcode   |  

Problem Description:

Given an array of n integers where n > 1, nums, return an array output such that output[i] is equal to the product of all the elements of nums except nums[i].

Solve it without division and in O(n).

For example, given [1,2,3,4], return [24,12,8,6].

Solutions:

To record everything’s product from left and from right and make a product.

Code in C++:

class Solution {
    public:
        vector<int> productExceptSelf(vector<int>& nums) {
               vector<int> left(nums.size(),1);
               for(int i=1;i<left.size();i++)
                   left[i]= left[i-1]*nums[i-1];
               vector<int> right(nums.size(),1);
               for(int i=right.size()-2;i>=0;i--)
                   right[i]=right[i+1]*nums[i+1];
               vector<int> res(nums.size(),1);
               for(int i=0;i<res.size();i++)
                   res[i]=left[i]*right[i];
               return res;
    }
};

Leetcode刷题记录67-Single Number III

发表于 2015-10-15   |   分类于 Leetcode   |  

Problem Description:

Given an array of numbers nums, in which exactly two elements appear only once and all the other elements appear exactly twice. Find the two elements that appear only once.

For example:

Given nums = [1, 2, 1, 3, 2, 5], return [3, 5].

Note:
The order of the result is not important. So in the above example, [5, 3] is also correct.
Your algorithm should run in linear runtime complexity. Could you implement it using only constant space complexity?

Solutions:

Using a set to store unique items.

Code in C++:

class Solution {
    public:
        vector<int> singleNumber(vector<int>& nums) {
            set<int> s;
            s.clear();
            for(int i=0;i<nums.size();i++)
                {
                    if(s.find(nums[i])!=s.end())
                        s.erase(nums[i]);
                    else 
                        s.insert(nums[i]);
                }
            vector<int> res;
            res.push_back(*s.begin());
            s.erase(*s.begin());
            res.push_back(*s.begin());
            return res;

    }
};
