Leetcode刷题记录81-Best Time to Buy and Sell Stock

发表于 2015-10-27   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Say you have an array for which the ith element is the price of a given stock on day i.

If you were only permitted to complete at most one transaction (ie, buy one and sell one share of the stock), design an algorithm to find the maximum profit.

Solutions:

This problem is very similar to This problem.Just change the array into the change array.

Code in C++:

class Solution {
    public:
        int maxProfit(vector<int>& prices) {
            if(prices.empty()) 
                return 0;
            int change[prices.size()];
            change[0]=0;
            for(int i=1;i<prices.size();i++)
                change[i]=prices[i]-prices[i-1];
            int res=INT_MIN;
               int sum=0;
            for(int i=0;i<prices.size();i++){
                sum= sum>=0? sum+change[i]:change[i];
                res=max(res,sum);
        }
            return res;
    }
};

Leetcode刷题记录80-Unique Paths

发表于 2015-10-27   |   分类于 Leetcode   |     |   阅读次数

Problem Descriptions:

A robot is located at the top-left corner of a m x n grid (marked ‘Start’ in the diagram below).

The robot can only move either down or right at any point in time. The robot is trying to reach the bottom-right corner of the grid (marked ‘Finish’ in the diagram below).

How many possible unique paths are there?

Solutions 1:

Using recursive call. It’s like the Climbing stairs problem.

Code in C++ for Solution 1:

class Solution {
    public:
        int uniquePaths(int m, int n) {
            if(m==1||n==1) return 1;
            else return uniquePaths(m-1,n)+uniquePaths(m,n-1);

    }
};

Solution 2:

Using a map to record the best solutions(using dynamic programming)

Code in C++ for Solution 2:

class Solution {
       public:
           int uniquePaths(int m, int n) {
           int a[m][n];
           for(int i=0;i<m;i++)
                a[i][n-1]=1;
           for(int i=0;i<n;i++)
               a[m-1][i]=1;
           for(int i=m-2;i>=0;i--)
               for(int j=n-2;j>=0;j--)
                   a[i][j]=a[i+1][j]+a[i][j+1];
           return a[0][0];
        }
    };

Leetcode刷题记录79-Find Minimum

发表于 2015-10-27   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Suppose a sorted array is rotated at some pivot unknown to you beforehand.

(i.e., 0 1 2 4 5 6 7 might become 4 5 6 7 0 1 2).

Find the minimum element.

You may assume no duplicate exists in the array.

Solutions:

Using binary search

Code in C++:

class Solution {
    public:
        int findMin(vector<int>& nums) {
            int begin =0;
            int end = nums.size()-1;
            int mid =begin+(end-begin)/2;
            if(nums[end]>nums[begin]) return nums[begin];
            while(end-begin>1)
                {
                    if(nums[mid-1]>nums[mid])
                        return nums[mid];
                    else if(nums[mid]<nums[begin])
                        {
                            end = mid ;
                            mid = begin + (end-begin)/2;
                        }
                    else if (nums[mid]>nums[begin])
                        {
                            begin = mid;
                            mid = begin + (end-begin)/2;
                        }
                }
            return min(nums[end],nums[begin]);

        }
};

Leetcode刷题记录78-Convert Sorted Array to Binary Search Tree

发表于 2015-10-25   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an array where elements are sorted in ascending order, convert it to a height balanced BST.

Solutions:

Do it recursively. use mid element to be the root.

Code in C++:

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
    public:
        TreeNode* sortedArrayToBST(vector<int>& nums) {
            vector<int> a;
            vector<int> b;
            int mid=nums.size()/2;
            for(int i=0;i<mid;i++)
            a.push_back(nums[i]);
            for(int i=mid+1;i<nums.size();i++)
            b.push_back(nums[i]);
            if(nums.size()==0) 
                return NULL;
            else if(nums.size()==1)
                {
                    TreeNode* t= new TreeNode(nums[0]);
                    return t;
                }
            else
                {
                       TreeNode* root =new TreeNode(nums[mid]);
                       root->left= sortedArrayToBST(a);
                       root->right=sortedArrayToBST(b);
                       return root;
                }
        }
    };

Leetcode刷题记录77-Maximum Subarray

发表于 2015-10-25   |   分类于 Leetcode   |     |   阅读次数

Problem Descriptions:

Find the contiguous subarray within an array (containing at least one number) which has the largest sum.

For example, given the array [−2,1,−3,4,−1,2,1,−5,4],
the contiguous subarray [4,−1,2,1] has the largest sum = 6.

Solutions:

Once the sum is less than 0, start from the current element.

Code in C++:

class Solution {
    public:
        int maxSubArray(vector<int>& nums) {
            int res=INT_MIN;
            int sum=0;
            for(int i=0;i<nums.size();i++){
                sum= sum>=0? sum+nums[i]:nums[i];
                res=max(res,sum);
            }
        return res;
    }
};

Leetcode刷题记录76-Integer to Roman

发表于 2015-10-25   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an integer, convert it to a roman numeral.

Input is guaranteed to be within the range from 1 to 3999.

Solutions:

Like coin pick problems, just use greedy algorithm

Code in C++:

class Solution {
    public:
        string intToRoman(int num) {
            string res="";
            int a[13]={1000,900,500,400,100,90,50,40,10,9,5,4,1};
            string x[13]={"M","CM","D","CD","C","XC","L","XL","X","IX","V","IV","I"};
            for(int i=0;i<13;i++)
                {
                    if(num>=a[i])
                        {
                            for(int j=0;j<num/a[i];j++)
                                res+=x[i];
                                num%=a[i];
                        }
                }
            return res;

        }
    };

Leetcode刷题记录75-Search Insert Position

发表于 2015-10-25   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a sorted array and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

You may assume no duplicates in the array.

Here are few examples.
[1,3,5,6], 5 → 2
[1,3,5,6], 2 → 1
[1,3,5,6], 7 → 4
[1,3,5,6], 0 → 0

Solutions:

Using binary search

Code in C++:

class Solution {
    public:
        int searchInsert(vector<int>& nums, int target) {
            int begin =0;
            int end = nums.size()-1;
            int mid = begin+(end-begin)/2;
            while(end-begin>1)
                    {
                    if(nums[mid]==target) 
                        return mid;
                    else if(nums[mid]>target)
                        {
                            end=mid;
                            mid=begin+(end-begin)/2;
                            continue;
                        }
                    else if(nums[mid]<target)
                        {
                            begin=mid;
                            mid=begin+(end-begin)/2;
                            continue;
                        }
                }
            if(target<=nums[begin])return begin;
            if(target>nums[end]) return end+1;
            return end;

    }
};

Leetcode刷题记录74-Unique Binary Search Tree

发表于 2015-10-25   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?

For example,
Given n = 3, there are a total of 5 unique BST’s.

1 3 3 2 1
\ / / / \ \
3 2 1 1 3 2
/ / \ \
2 1 2 3

Solutions:

A unique binary search tree using i as root in n nodes are (0~i-1) in the left and (i+1~n) in the right. So we can do a dynamic programming.

Updates:
You can see from here . This kind of number is called “Catalan number”.

Code in C++:

class Solution {
    public:
        int numTrees(int n) {
            int res[n+1]={0};
            res[0]=1;
            res[1]=1;
            for(int i=2;i<=n;i++)
                    for(int j=0;j<i;j++)
                      res[i]+=res[j]*res[i-1-j];
            return res[n];
    }
};

Leetcode刷题记录73-Missing Number

发表于 2015-10-24   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an array containing n distinct numbers taken from 0, 1, 2, …, n, find the one that is missing from the array.

For example,
Given nums = [0, 1, 3] return 2.

Note:
Your algorithm should run in linear runtime complexity. Could you implement it using only constant extra space complexity?

Solution 1:

If we have O(n) extra space. we can mark every appeard element.

Code in C++ for Solution 1:

class Solution {
    public:
        int missingNumber(vector<int>& nums) {
            bool flag[nums.size()+1];
            for(int i=0;i<nums.size();i++) 
                flag[i]=false;
            for(int i=0;i<nums.size();i++)
                flag[nums[i]]= true;
            for(int i=0;i<nums.size();i++)
                if(!flag[i]) return i;
            return nums.size();

    }
};

Solution 2:

We can add the total number and minus every element.

Code in C++ for Solution 2:

class Solution {
    public:
        int missingNumber(vector<int>& nums) {
            int res=nums.size();
            for(int i=0;i<nums.size();i++)
                res += (i-nums[i]);
            return res;

    }
};

Solution 3:

Use Bit Manipulation(which I don’t know very well)

Code in C++ for Solution 3:

class Solution {
    public:
        int missingNumber(vector<int>& nums) {
            int res = 0;
            for (int i = 0; i < nums.size(); ++i) {
                res ^= (i + 1) ^ nums[i];
        }
            return res;
    }
};

Leetcode刷题记录72-Populating Next Right Pointers in Each Node

发表于 2015-10-24   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a binary tree

struct TreeLinkNode {
  TreeLinkNode *left;
  TreeLinkNode *right;
  TreeLinkNode *next;
}

Populate each next pointer to point to its next right node. If there is no next right node, the next pointer should be set to NULL.

Initially, all next pointers are set to NULL.

Note:

You may only use constant extra space.
You may assume that it is a perfect binary tree (ie, all leaves are at the same level, and every parent has two children).
For example,
Given the following perfect binary tree,
1
/ \
2 3
/ \ / \
4 5 6 7
After calling your function, the tree should look like:
1 -> NULL
/ \
2 -> 3 -> NULL
/ \ / \
4->5->6->7 -> NULL

Solutions:

It’s very like This Problem

Code in C++:

 /**
 * Definition for binary tree with next pointer.
 * struct TreeLinkNode {
 *  int val;
 *  TreeLinkNode *left, *right, *next;
 *  TreeLinkNode(int x) : val(x), left(NULL), right(NULL), next(NULL) {}
 * };
 */
class Solution {
    public:
        void connect(TreeLinkNode *root) {
            queue<TreeLinkNode*> a;
            if(root)
                {
                    TreeLinkNode* pre=NULL;
                    a.push(root);
                    a.push(NULL);
                    while(!a.empty())
                        {
                            TreeLinkNode* t=a.front();
                            a.pop();
                            if(t)
                                {
                                    if(t->left)
                                        a.push(t->left);
                                    if(t->right)
                                        a.push(t->right);
                                    if(pre) 
                                        pre->next = t;
                                    pre=t;                                 
                                }
                            else{
                                    if(!a.empty())
                                        {
                                            a.push(NULL);
                                            if(pre) 
                                                pre->next=NULL;
                                            pre=NULL;
                                        }
                                }

                        }
                }

        }
    };
