Leetcode刷题记录54-Rotate Array

发表于 2015-10-13   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Rotate an array of n elements to the right by k steps.

For example, with n = 7 and k = 3, the array [1,2,3,4,5,6,7] is rotated to [5,6,7,1,2,3,4].

Solutions:

Pay attention to the situation that k might > nums.size()

Code in C++:

class Solution {
    public:
        void rotate(vector<int>& nums, int k) {
            vector<int> tem;
            k%=nums.size();
            for(int i=nums.size()-k;i<=nums.size()-1;i++)
                {
                    tem.push_back(nums[i]);
                }
            for(int j=nums.size()-1;j>=k;j--)
                {
                       nums[j]=nums[j-k];
                }
            for(int m=0;m<tem.size();m++)
                {
                    nums[m]=tem[m];
                }


        }
};

Leetcode刷题记录53-Implement Str

发表于 2015-10-13   |   分类于 Leetcode   |     |   阅读次数

Problem Descriptions:

Returns the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Solutions:

Use two pointers

Code in C++:

class Solution {
    public:
        int strStr(string haystack, string needle) {
            if(needle=="") 
                return 0;
            if(haystack.length()<needle.length()) 
                return -1;
            int i=0;
            int j=needle.length();
            for(;i<haystack.length()-needle.length()+1;i++)
                {
                    for(int k=0;k<j;k++)
                           {
                            if(haystack[i+k]!=needle[k]) 
                                break;
                            if(k==j-1) 
                                return i;
                        }

                }
            return -1;
        }
    };

Leetcode刷题记录52-Valid Palindrome

发表于 2015-10-12   |   分类于 Leetcode   |     |   阅读次数

Problem Descriptions:

Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

For example,
“A man, a plan, a canal: Panama” is a palindrome.
“race a car” is not a palindrome.

Note:
Have you consider that the string might be empty? This is a good question to ask during an interview.

For the purpose of this problem, we define empty string as valid palindrome.

Solutions:

Just simplify the string and do the decision.

Code in C++:

class Solution {
    public:
        bool isPalindrome(string s) {
               string plain ="";
            for(int i =0;i<s.length();i++)
                {
                    if(s[i]>='a'&&s[i]<='z')
                        plain+=s[i];
                    else if(s[i]>='A'&&s[i]<='Z')
                        plain+=(char)(s[i]-'A'+'a');
                    else if(s[i]>='0'&&s[i]<='9')
                        plain+=s[i];
                    else continue;
                }
            if(s.length()<=1) 
                return true;
            for(int i=0;i<plain.length()/2+1;i++)
            if(plain[i]!=plain[plain.length()-1-i]) 
                return false;
            return true;

    }
};

Leetcode刷题记录51-Palindrome Linked List

发表于 2015-10-12   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a singly linked list, determine if it is a palindrome.

Could you do it in O(n) time and O(1) space?

Solutions:

reverse the second part and check if it is the same as the first half.

Code in C++:

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
    public:
        ListNode* reverseList(ListNode* head) {
        if(head == NULL) 
            return NULL;
        if(head->next == NULL) 
            return head;
        ListNode* p;
        ListNode* q;
        p = head->next;
        while(p->next!=NULL)
            {
                q=p->next;
                p->next = q->next;
                q->next = head->next;
                head->next = q;
            }
        p->next = head;
        head = p->next->next;
        p->next->next = NULL;

        return head;

    }
    bool isPalindrome(ListNode* head) {
        if(!head||!head->next) 
            return true;
           ListNode* first = head;
           ListNode* second = head;
          ListNode* counthelper=head;
           int secondindex=0;
           int count=0;
           while(counthelper!=NULL)
               {
                   counthelper=counthelper->next;
                   count++;
               }
           if(count%2==0)
               secondindex=count/2;
           else 
               secondindex =count/2+1;
           for(int i=0;i<secondindex;i++)
               second=second->next;
           second=reverseList(second);
           while(second!=NULL)
               {
                   if(first->val!=second->val)
                       return false;
                   first=first->next;
                   second=second->next;
               }
           return true;

    }
};

Leetcode刷题记录50-Nim Game

发表于 2015-10-12   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

You are playing the following Nim Game with your friend: There is a heap of stones on the table, each time one of you take turns to remove 1 to 3 stones. The one who removes the last stone will be the winner. You will take the first turn to remove the stones.

Both of you are very clever and have optimal strategies for the game. Write a function to determine whether you can win the game given the number of stones in the heap.

For example, if there are 4 stones in the heap, then you will never win the game: no matter 1, 2, or 3 stones you remove, the last stone will always be removed by your friend.

Solutions:

This is a simple math problem. The first will win only if the number of total stones have anything left if is divided by 4.

Code in C++:

class Solution {
    public:
        bool canWinNim(int n) {
            return n%4>0;
    }
};

Problem Description:

Reverse digits of an integer.

Example1: x = 123, return 321
Example2: x = -123, return -321

click to show spoilers.

Have you thought about this?
Here are some good questions to ask before coding. Bonus points for you if you have already thought through this!

If the integer’s last digit is 0, what should the output be? ie, cases such as 10, 100.

Did you notice that the reversed integer might overflow? Assume the input is a 32-bit integer, then the reverse of 1000000003 overflows. How should you handle such cases?

For the purpose of this problem, assume that your function returns 0 when the reversed integer overflows.

Solutions:

A simple math task

Code in C++:

class Solution {
    public:
        int reverse(int x) {
               int res = 0;
            while(x!=0)
                {
                    if(res>INT_MAX/10||res<INT_MIN/10) 
                        return 0;
                    else
                        {
                            res= res*10+x%10;
                            x/=10;
                        }
                }
            return res;
        }
};

Leetcode刷题记录48-Add Binary

Problem Description:

Given two binary strings, return their sum (also a binary string).

For example,
a = “11”
b = “1”
Return “100”.

Solutions:

Handle that from back to front;

Code in C++:

class Solution {
    public:
        string addBinary(string a, string b) {
            string res;
            int carry=0;
                int i=a.length()-1;
            int j=b.length()-1;
                for(;i>=0||j>=0;i--,j--)
                {
                    int aa,bb;
                       if(i>=0)
                           aa=(a[i]-'0');
                    else 
                        aa=0;
                    if(j>=0)
                        bb=(b[j]-'0');
                    else 
                        bb=0;
                    int current=aa+bb+carry;
                    char bit;
                    switch(current)
                        {
                                case 0: bit='0';carry=0;break;
                                 case 1: bit='1';carry=0;break;
                                case 2: bit='0';carry=1;break;
                              case 3: bit='1';carry=1;break;
                              default:break;
                        }
                    res=bit+res;
                }
            if(carry)
                {
                    string s="1";
                    return s+res;
                }
            return res;
    }
};

Leetcode刷题记录47-Count and Say

发表于 2015-10-11   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

The count-and-say sequence is the sequence of integers beginning as follows:
1, 11, 21, 1211, 111221, …

1 is read off as “one 1” or 11.
11 is read off as “two 1s” or 21.
21 is read off as “one 2, then one 1” or 1211.
Given an integer n, generate the nth sequence.

Note: The sequence of integers will be represented as a string.

Solutions:

Do it very carefully with borders

Code in C++:

class Solution {
    public:
        string int2str(int num) 
            { 
                if(num == 0) 
                    return "0"; 

                string str = ""; 

                int num_ = num > 0 ? num : -1 * num; 

                while(num_) 
                    { 
                        str = (char)(num_ % 10 + 48) + str; 
                        num_ /= 10; 
                    } 

                if(num < 0) 
                    str = "-" + str; 
                    return str; 
            }     
        string countAndSay(int n) {
                if(n<1) 
                    return "1";
                vector<string> res;
                res.push_back("1");
                int count=1;
                char current=' ';
                string temp="";
                for(int i=1;i<n;i++)
                    {
                           temp="";
                          count=0;
                        current=res[i-1][0];
                        for(int j=0;j<res[i-1].length();j++)
                            {
                                if(res[i-1][j]==current)
                                    count++;

                                else
                                    {
                                        temp+=int2str(count);
                                           temp+=current;
                                        current=res[i-1][j];
                                        count=1;
                                    }
                                if(j==res[i-1].length()-1)
                                    {
                                           temp+=int2str(count);
                                        temp+=current;

                                    }
                            }
                        res.push_back(temp);

                    }
                   return res[n-1]; 
        }
    };

Leetcode刷题记录46-Valid Parentheses

发表于 2015-10-11   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a string containing just the characters ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ and ‘]’, determine if the input string is valid.

The brackets must close in the correct order, “()” and “()[]{}” are all valid but “(]” and “([)]” are not.

Solutions:

Very classic stack problem.

Code in C++:

class Solution {
    public:
        bool isValid(string s) {
               if(s.length()==0) 
                    return true;
            vector<char> stack;
            stack.clear();
            for(int i =0;i<s.length();i++)
                {

                      switch(s[i])
                          {
                              case '(':
                                          stack.push_back(s[i]);break;
                              case '[':
                                          stack.push_back(s[i]);break;
                              case '{':
                                          stack.push_back(s[i]);break;
                             case ')':
                                          if(stack.size()>0&&stack.back()=='(') 
                                              stack.pop_back(); 
                                          else 
                                              return false;
                                              break;
                              case ']':
                                          if(stack.size()>0&&stack.back()=='[') 
                                              stack.pop_back(); 
                                          else 
                                              return false;
                                              break;
                              case '}':
                                          if(stack.size()>0&&stack.back()=='{') 
                                              stack.pop_back(); 
                                          else 
                                              return false;
                                              break;
                              default: 
                                          return false;
                          }



            }  
            return stack.size()==0;
    }
};

Leetcode刷题记录45-Remove Nth Node From End Of List

发表于 2015-10-11   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a linked list, remove the nth node from the end of list and return its head.

For example,

Given linked list: 1->2->3->4->5, and n = 2.

After removing the second node from the end, the linked list becomes 1->2->3->5.
Note:
Given n will always be valid.
Try to do this in one pass.

Solutions:

Use two pointers

Code in C++:

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
    public:
        ListNode* removeNthFromEnd(ListNode* head, int n) {
               ListNode *pre=head;
            ListNode *b=head->next;
            if(!b) 
                return NULL;
            while(pre&&b)
                {
                for(int i = 0;i<n-1;i++) b=b->next;
                if (!b) 
                    return head->next;
                else
                {
                    b=b->next;
                    if(!b)
                        {
                            pre->next=pre->next->next;
                             return head;
                            }
                    else
                        {
                            pre=pre->next;
                            b=pre->next;
                        }
                }



                }
            return head;
        }
    };
