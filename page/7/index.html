Leetcode刷题记录27-Add Digits

发表于 2015-10-09   |   分类于 Leetcode   |  

Problem Description:

Given a non-negative integer num, repeatedly add all its digits until the result has only one digit.

For example:

Given num = 38, the process is like: 3 + 8 = 11, 1 + 1 = 2. Since 2 has only one digit, return it.

Follow up:

Could you do it without any loop/recursion in O(1) runtime?

Solutions:

Write a few results and find the clue.

Code in C++:

class Solution {
    public:
        int addDigits(int num) {
            if(num<10) 
                return num;
            return num%9==0?9:num%9;
        }
};

Leetcode刷题记录26-Single Number

发表于 2015-10-08   |   分类于 Leetcode   |  

Problem Description:

Given an array of integers, every element appears twice except for one. Find that single one.

Solutions:

Using a set to store the unique ones, delete when duplicates are found.

Code in C++:

class Solution {
    public:
        int singleNumber(vector<int>& nums) {
            set<int> set;
            set.clear();
            for(int i =0;i<nums.size();i++)
                {
                    if(set.find(nums[i])!=set.end())
                        set.erase(nums[i]);
                    else 
                        set.insert(nums[i]);
                }
            return *set.begin();
        }
};

Leetcode刷题记录25-Isomorphic Strings

发表于 2015-10-08   |   分类于 Leetcode   |  

Problem Description:

Given two strings s and t, determine if they are isomorphic.

Two strings are isomorphic if the characters in s can be replaced to get t.

All occurrences of a character must be replaced with another character while preserving the order of characters. No two characters may map to the same character but a character may map to itself.

For example,
Given “egg”, “add”, return true.

Given “foo”, “bar”, return false.

Given “paper”, “title”, return true.

Solutions:

It’s the same as This Problem.And it’s simpler.

Code in C++:

class Solution {
    public:
        bool isIsomorphic(string s, string t) {            
            string pattern = s;
            int len = pattern.length();
            string a = t;
            std::map<char,char> map;
            std::map<char,char> map2;
            std::map<char,char>::iterator it;
            std::map<char,char>::iterator it2;

            for(int i =0;i<len;i++)
                {
                    it = map.find(pattern[i]);
                    it2 = map2.find(a[i]);
                    if (it != map.end()|| it2 !=map2.end())
                        {
                            if(a[i]!=map[pattern[i]])
                                return false;
                            if(map[map2[a[i]]]!=a[i]) return false;
                        }
                    else
                        {
                            map[pattern[i]]=a[i];
                            map2[a[i]]=pattern[i];  
                        }
                }
            return true;
        }
    };

Leetcode刷题记录24-Longest Common Prefix

发表于 2015-10-08   |   分类于 Leetcode   |  

Problem Description:

Write a function to find the longest common prefix string amongst an array of strings.

Solutions:

A very simple task.

Code in C++:

class Solution {
    public:
        string longestCommonPrefix(vector<string>& strs) {
            if(strs.size()==0) 
                return "";
            string res = "";
            string restem =res+strs[0][0];
            while(res.length()<strs[0].length())
                {
                    for(int i = 0;i< strs.size();i++)
                        {
                            for(int j = 0;j<restem.length();j++)
                                {
                                    if(strs[i][j]!=restem[j]) 
                                        return res;
                                }
                        }
                    res = restem;
                    restem+=strs[0][restem.length()];
                }

            return res;

        }
    };

Leetcode刷题记录23-Remove Linked List Elements

发表于 2015-10-08   |   分类于 Leetcode   |  

Problem Description:

Remove all elements from a linked list of integers that have value val.

Example
Given: 1 –> 2 –> 6 –> 3 –> 4 –> 5 –> 6, val = 6
Return: 1 –> 2 –> 3 –> 4 –> 5

Solutions:

Pay attention when first few elements is to be deleted

Code in C++:

/**
* Definition for singly-linked list.
* struct ListNode {
*     int val;
*     ListNode *next;
*     ListNode(int x) : val(x), next(NULL) {}
* };
*/
class Solution {
    public:
        ListNode* removeElements(ListNode* head, int val) {
            if(head == NULL) 
                return NULL;
            while(head->val == val) 
                {
                    head=head->next;
                    if(head == NULL) 
                        return head;
                }
            ListNode* a = head;
            while(a->next!= NULL&&a!=NULL)
                {
                    if(a->next->val == val)
                        {
                            ListNode * b= a->next;
                            a->next = b->next;
                        }
                    else
                        {
                            a=a->next;
                        }
                }
            return head;

        }
};

Problem Description:

Given a pattern and a string str, find if str follows the same pattern.

Examples:
pattern = “abba”, str = “dog cat cat dog” should return true.
pattern = “abba”, str = “dog cat cat fish” should return false.
pattern = “aaaa”, str = “dog cat cat dog” should return false.
pattern = “abba”, str = “dog dog dog dog” should return false.
Notes:
patterncontains only lowercase alphabetical letters, and str contains words separated by a single space. Each word in str contains only lowercase alphabetical letters.
Both pattern and str do not have leading or trailing spaces.
Each letter in pattern must map to a word with length that is at least 1.

Solutions:

use two hashtables to store the pairs.

Code in C++:

class Solution {

        vector<string> stringTovec(string s){
            vector<string> res;

            string temp="";
            for(int i=0;i<len;++i)
                {
                    if(s[i]==' ')
                        {
                            res.push_back(temp);
                            temp="";
                        }
                    else
                        temp+=s[i];
                }
            res.push_back(temp);
            return res;
        }

        bool wordPattern(string pattern, string str) {
            if(pattern.length()==0||str.length()==0) 
                return false;
            int len = pattern.length();
            vector<string> a = stringTovec(str);
            if(a.size()!=len) return false;

            std::map<char,string> map;
            std::map<string,char> map2;
            std::map<char,string>::iterator it;
            std::map<string ,char>::iterator it2;

            for(int i =0;i<len;i++)
                {
                    it = map.find(pattern[i]);
                    it2 = map2.find(a[i]);
                    if (it != map.end()|| it2 !=map2.end())
                        {
                            if(a[i]!=map[pattern[i]])
                                return false;
                            if(map[map2[a[i]]]!=a[i]) 
                                return false;
                        }
                    else
                        {
                            map[pattern[i]]=a[i];
                            map2[a[i]]=pattern[i];  
                        }
                }
            return true;


        }
};

Leetcode刷题记录21-Rectangular Area

发表于 2015-10-08   |   分类于 Leetcode   |  

Problem Description:

Find the total area covered by two rectilinear rectangles in a 2D plane.

Each rectangle is defined by its bottom left corner and top right corner as shown in the figure.

Rectangle Area
Assume that the total area is never beyond the maximum possible value of int.

Solutions:

Easy math problem. remember to cut off repeat part.

Code in C++:

class Solution {
    public:
        int computeArea(int A, int B, int C, int D, int E, int F, int G, int H) {
            int coverleft = A>E?A:E;
            int coverright = C<G?C:G;
            int covertop = D>H?H:D;
            int coverbot = F>B?F:B;

            int coverarea = 0;
            if(coverleft<=coverright&&covertop>=coverbot)
                {
                    coverarea=(coverright-coverleft)*(covertop-coverbot);

                }
            return(C-A)*(D-B)+(G-E)*(H-F)-coverarea;

        }
    };

Leetcode刷题记录20-Contains Duplicate II

发表于 2015-10-07   |   分类于 Leetcode   |  

Problem Description:

Given an array of integers and an integer k, find out whether there are two distinct indices i and j in the array such that nums[i] = nums[j] and the difference between i and j is at most k.

Solutions:

Using a moving window of at most size k , find there’s duplicats in the window.

Code in C++:

class Solution {
    public:
        bool containsNearbyDuplicate(vector<int>& nums, int k) {
            if(nums.size()<=1||k<=0) 
                return false;
            else
                {
                    set<int> a;
                    for(int i =0;i<nums.size();i++)
                        {
                            if(a.size()>k)
                                {
                                    a.erase(nums[i-k-1]);
                                }
                            if(a.find(nums[i])!=a.end())
                                return true;

                            a.insert(nums[i]);
                        }
                    return false;
                }
        }
    };

Leetcode刷题记录19-Length Of Last Word

发表于 2015-10-07   |   分类于 Leetcode   |  

Problem Description:

Given a string s consists of upper/lower-case alphabets and empty space characters ‘ ‘, return the length of last word in the string.

If the last word does not exist, return 0.

Note: A word is defined as a character sequence consists of non-space characters only.

For example,
Given s = “Hello World”,
return 5.

Solutions:

just simple manipulations, remember to take care of special cases

Code in C++:

class Solution {
    public:
        int lengthOfLastWord(string s) {
            if(s.length() == 0) 
                return 0;
            int count = 0;
            int flag= s.length()-1;
            for(int i = s.length()-1;i>=0;i--)
                {
                    if(s[i]!=' ') {flag = i;break;}
                }

            while(s[flag-count]!=' '&&count<=flag)
                {
                    count++;
                }
            return count;
        }
    };

Leetcode刷题记录18-Merge Sorted Array

Problem Description:

Given two sorted integer arrays nums1 and nums2, merge nums2 into nums1 as one sorted array.

Note:
You may assume that nums1 has enough space (size that is greater or equal to m + n) to hold additional elements from nums2. The number of elements initialized in nums1 and nums2 are m and n respectively.

Solutions:

Insert from back.

Code in C++:

class Solution {
    public:
        void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
            if(m == 0)
                {
                    for(int i =0;i<n;i++)
                        nums1[i]=nums2[i];
                }
            else
                {
                    int j = m+n-1;
                    m--;
                    n--;
                    for(;j>=0&&m>=0&&n>=0;j--)
                        {
                            if(nums1[m]>=nums2[n])
                                {
                                    nums1[j]=nums1[m--];
                                }
                            else if(nums1[m]<nums2[n])
                                {
                                    nums1[j]=nums2[n--];
                                }
                        }
                    if(n>=0)
                        {
                            for(int i = 0;i<=n;i++)
                                nums1[i]=nums2[i];
                        } 
                }


            }
    };
