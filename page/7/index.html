Leetcode刷题记录33-Majority Element

发表于 2015-10-10   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.

You may assume that the array is non-empty and the majority element always exist in the array.

Solutions:

Every other element will cancel the majority once. The left one is the majority.

Code in C++:

class Solution {
    public:
        int majorityElement(vector<int>& nums) {
            int res = nums[0];
            int count = 1;
            for(int i =1;i<nums.size();i++)
                {    
                    if(nums[i]==res) 
                        count++;
                    else if(nums[i]!=res) 
                        count--;
                    if(count==0) 
                        {
                            res = nums[i];
                            count=1;
                        }
                }
            return res;
        }
    };

Leetcode刷题记录32-Remove Duplicates from Sorted List

发表于 2015-10-09   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a sorted linked list, delete all duplicates such that each element appear only once.

For example,
Given 1->1->2, return 1->2.
Given 1->1->2->3->3, return 1->2->3.

Solutions:

Simple task

Code in C++:

/**
* Definition for singly-linked list.
* struct ListNode {
*     int val;
*     ListNode *next;
*     ListNode(int x) : val(x), next(NULL) {}
* };
*/
class Solution {
    public:
        ListNode* deleteDuplicates(ListNode* head) {
            if(head==NULL||head->next==NULL) 
                return head;
            else
                {
                    ListNode*a = head;      
                    ListNode*b = head->next;
                    while(a!=NULL&&b!=NULL)
                        {
                            if(a->val==b->val)
                                {
                                    a->next=b->next;
                                    b=b->next;
                                }
                            else
                                {
                                    a= a->next;
                                    b=b->next;
                                }

                        }
                    return head; 
                }

        }
    };

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

A -> 1

...

AB -> 28

Solutions:

Easy task

Code in C++:

class Solution {
    public:
        int toInt(char a)
            {
                if(a>='A'&&a<='Z')
                    return (int)(a-'A'+1);
                else 
                    return 0;
            }
int titleToNumber(string s) {
    int res = 0;
    for(int i =0;i<s.length();i++)
        {
            res+= (pow(26,s.length()-i-1)*toInt(s[i])); 
        }
    return res;

    }
};

Leetcode刷题记录30-Same Tree

发表于 2015-10-09   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given two binary trees, write a function to check if they are equal or not.

Two binary trees are considered equal if they are structurally identical and the nodes have the same value.

Solutions:

Using Depth First Search.

Code in C++:

/**
* Definition for a binary tree node.
* struct TreeNode {
*     int val;
*     TreeNode *left;
*     TreeNode *right;
*     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
* };
*/
class Solution {
    public:
        bool isSameTree(TreeNode* p, TreeNode* q) {
            if(!p&&!q) 
                return true;
            if((p&&!q)||(q&&!p)) 
                return false;
            return isSameTree(p->left,q->left)&&isSameTree(p->right,q->right)&&(p->val==q->val);
        }
};

Leetcode刷题记录29-Delete Node in a Linked List

发表于 2015-10-09   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Write a function to delete a node (except the tail) in a singly linked list, given only access to that node.

Supposed the linked list is 1 -> 2 -> 3 -> 4 and you are given the third node with value 3, the linked list should become 1 -> 2 -> 4 after calling your function.

Solutions:

Simply use this link to point to the next node.

Code in C++:

/**
* Definition for singly-linked list.
* struct ListNode {
*     int val;
*     ListNode *next;
*     ListNode(int x) : val(x), next(NULL) {}
* };
*/
class Solution {
    public:
        void deleteNode(ListNode* node) {
            node->val = node->next->val;
            node->next = node->next->next;
        }
};

Leetcode刷题记录28-Maximum Depth of Bianary Tree

发表于 2015-10-09   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Solutions:

So easy…

Code in C++:

/**
* Definition for a binary tree node.
* struct TreeNode {
*     int val;
*     TreeNode *left;
*     TreeNode *right;
*     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
* };
*/
class Solution {
    public:
        int maxDepth(TreeNode* root) {
        if(root==NULL) 
            return 0;
        return max(maxDepth(root->left)+1,maxDepth(root->right)+1);
    }
};

Leetcode刷题记录27-Add Digits

发表于 2015-10-09   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a non-negative integer num, repeatedly add all its digits until the result has only one digit.

For example:

Given num = 38, the process is like: 3 + 8 = 11, 1 + 1 = 2. Since 2 has only one digit, return it.

Follow up:

Could you do it without any loop/recursion in O(1) runtime?

Solutions:

Write a few results and find the clue.

Code in C++:

class Solution {
    public:
        int addDigits(int num) {
            if(num<10) 
                return num;
            return num%9==0?9:num%9;
        }
};

Leetcode刷题记录26-Single Number

发表于 2015-10-08   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an array of integers, every element appears twice except for one. Find that single one.

Solutions:

Using a set to store the unique ones, delete when duplicates are found.

Code in C++:

class Solution {
    public:
        int singleNumber(vector<int>& nums) {
            set<int> set;
            set.clear();
            for(int i =0;i<nums.size();i++)
                {
                    if(set.find(nums[i])!=set.end())
                        set.erase(nums[i]);
                    else 
                        set.insert(nums[i]);
                }
            return *set.begin();
        }
};

Leetcode刷题记录25-Isomorphic Strings

发表于 2015-10-08   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given two strings s and t, determine if they are isomorphic.

Two strings are isomorphic if the characters in s can be replaced to get t.

All occurrences of a character must be replaced with another character while preserving the order of characters. No two characters may map to the same character but a character may map to itself.

For example,
Given “egg”, “add”, return true.

Given “foo”, “bar”, return false.

Given “paper”, “title”, return true.

Solutions:

It’s the same as This Problem.And it’s simpler.

Code in C++:

class Solution {
    public:
        bool isIsomorphic(string s, string t) {            
            string pattern = s;
            int len = pattern.length();
            string a = t;
            std::map<char,char> map;
            std::map<char,char> map2;
            std::map<char,char>::iterator it;
            std::map<char,char>::iterator it2;

            for(int i =0;i<len;i++)
                {
                    it = map.find(pattern[i]);
                    it2 = map2.find(a[i]);
                    if (it != map.end()|| it2 !=map2.end())
                        {
                            if(a[i]!=map[pattern[i]])
                                return false;
                            if(map[map2[a[i]]]!=a[i]) return false;
                        }
                    else
                        {
                            map[pattern[i]]=a[i];
                            map2[a[i]]=pattern[i];  
                        }
                }
            return true;
        }
    };

Leetcode刷题记录24-Longest Common Prefix

发表于 2015-10-08   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Write a function to find the longest common prefix string amongst an array of strings.

Solutions:

A very simple task.

Code in C++:

class Solution {
    public:
        string longestCommonPrefix(vector<string>& strs) {
            if(strs.size()==0) 
                return "";
            string res = "";
            string restem =res+strs[0][0];
            while(res.length()<strs[0].length())
                {
                    for(int i = 0;i< strs.size();i++)
                        {
                            for(int j = 0;j<restem.length();j++)
                                {
                                    if(strs[i][j]!=restem[j]) 
                                        return res;
                                }
                        }
                    res = restem;
                    restem+=strs[0][restem.length()];
                }

            return res;

        }
    };
