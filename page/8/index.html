Leetcode刷题记录17-Happy Number

发表于 2015-10-07   |   分类于 Leetcode   |  

Problem Description:

Write an algorithm to determine if a number is “happy”.

A happy number is a number defined by the following process: Starting with any positive integer, replace the number by the sum of the squares of its digits, and repeat the process until the number equals 1 (where it will stay), or it loops endlessly in a cycle which does not include 1. Those numbers for which this process ends in 1 are happy numbers.

Example: 19 is a happy number

12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1

Solutions:

Make a hashtable to store every step’s result. Once you find the result in the hashtable, return if it’s equal to 1

Code in C++:

class Solution {
    public:
        int theSum(int n)
            {
                int result=0;
                while(n>=1)
                    {
                        result+=(n%10)*(n%10);
                        n/=10;
                    }
                return result;
            }
        bool isHappy(int n) {
            vector<int> a;
            while(1)
                {
                    std::vector<int>::iterator it;
                    it = find (a.begin(), a.end(), theSum(n));
                    if (it != a.end())
                        {
                            return(theSum(n) == 1);
                        }
                    else
                        {
                            a.push_back(theSum(n));
                            n=theSum(n);
                        }
                }

            }
    };

Leetcode刷题记录16-Remove Duplicates from Sorted Array II

发表于 2015-10-07   |   分类于 Leetcode   |  

Problem Description:

Follow up for “Remove Duplicates”:
What if duplicates are allowed at most twice?

For example,
Given sorted array nums = [1,1,1,2,2,3],

Your function should return length = 5, with the first five elements of nums being 1, 1, 2, 2 and 3. It doesn’t matter what you leave beyond the new length.

Solutions:

Use some flag to mark if a number has been kept for once.

Code in C++:

class Solution {
    public:
        int removeDuplicates(vector<int>& nums) {
            if (nums.size() == 0) 
                return 0;
            else if(nums.size() == 1) 
                return 1;
            else
                {
                    int count = 1;
                    int temp = nums[0];
                    bool flag = false;
                    for(int i = 1;i<nums.size();i++)
                        {
                            if(nums[i]!= temp) 
                                {   
                                    temp = nums[i];
                                    nums[count++]=nums[i];
                                    flag= false;

                                }
                            else if(nums[i] == temp) 
                                {
                                    if(!flag)
                                        {
                                            nums[count++]=nums[i];
                                            flag = true;
                                        }
                                }

                        }
                    return count;
                }    
        }
    };

Leetcode刷题记录15-Pascal's Triangle II

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

Given an index k, return the kth row of the Pascal’s triangle.

For example, given k = 3,
Return [1,3,3,1].

Note:
Could you optimize your algorithm to use only O(k) extra space?

Solutions:

Only use two arrays, one to store the answer and the other to store the previous answer.

Code in C++:

class Solution {
    public:
        vector<int> getRow(int rowIndex) {
        vector<int> last;
        last.push_back(1);
        if(rowIndex == 0) 
            return last;
        last.push_back(1);
        if(rowIndex == 1) 
            return last;
        if(rowIndex > 1)
            {
                vector<int> result;

                for(int i =1;i<rowIndex;i++)
                    {
                        result.clear();
                        result.push_back(1);
                        for(int i =1;i<last.size();i++)
                        result.push_back(last[i]+last[i-1]);    
                        result.push_back(1);
                        last=result;
                    }
                return result;
            }
        }
    };

Leetcode刷题记录14-House Robber

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed, the only constraint stopping you from robbing each of them is that adjacent houses have security system connected and it will automatically contact the police if two adjacent houses were broken into on the same night.

Given a list of non-negative integers representing the amount of money of each house, determine the maximum amount of money you can rob tonight without alerting the police.

Solution 1:

Use recursive algorithms. That will take so much time,and will not be accepted;

Code in C++ for Solution 1:

class Solution {
    public:
        int max(vector<int>& a, int p)
            {
                if(p == 0) return 0;
                if(p == 1) return a[0];
                if(p == 2) return a[0]>a[1]? a[0]:a[1];
                else
                    {
                        if(max(a,p-2)+a[p-1]>max(a,p-1)) 
                            return max(a,p-2)+a[p-1];
                        else 
                            return max(a,p-1);
                    }
            }
        int rob(vector<int>& nums) {
            return max(nums,nums.size());

            }
};

Solution 2:

Use Dynamic Programming, which will use an array to keep every best solution for each step.

Code in C++ for Solution 2:

class Solution {
    public:
        int rob(vector<int>& nums) {
            if(nums.size()==0) 
                return 0;
            if(nums.size()==1) 
                return nums[0];
            if(nums.size()==2) 
                return nums[0]>nums[1]?nums[0]:nums[1];
            if(nums.size()>=3)
                {
                    vector<int> solutions;
                    solutions.push_back(nums[0]);
                    solutions.push_back(nums[0]>nums[1]?nums[0]:nums[1]);
                    for(int i = 2; i < nums.size();i++)
                        if(solutions[i-2]+nums[i]>solutions[i-1])
                            solutions.push_back(solutions[i-2]+nums[i]);
                        else
                            solutions.push_back(solutions[i-1]);

                    return solutions.back();
                }

        }
    };

Leetcode刷题记录13-Pascal's Triangle

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

Given numRows, generate the first numRows of Pascal’s triangle.

For example, given numRows = 5,
Return

[
[1],
[1,1],
[1,2,1],
[1,3,3,1],
[1,4,6,4,1]
]

Solutions:

For every element after the second line, we can see that a[i][j]=a[i-1][j-1]+a[i-1][j], finally put a 1 at the back of the row.

Code in C++:

class Solution {
    public:
        vector<vector<int>> generate(int numRows) {
        vector<vector<int>> a;
        if(numRows == 0) return a;
        vector<int> last = {1};
        a.push_back(last);
        if(numRows == 1) 
            return a;
        last.push_back(1);
        a.push_back(last);
        if(numRows == 2) 
            return a;
        if(numRows>=3)
            {
                for(int i =2;i< numRows;i++)
                    {
                        last.clear();
                        last.push_back(1);
                        for(int j = 1; j < i; j++)
                            last.push_back(a[i-1][j-1]+a[i-1][j]);
                        last.push_back(1);
                        a.push_back(last);
                    }
                return a;   
            }


        }
    };

Leetcode刷题记录12-Plus One

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

Given a non-negative number represented as an array of digits, plus one to the number.

The digits are stored such that the most significant digit is at the head of the list.

Solutions:

It’s a easy task, just pay attention to all possible cases.

Code in C++:

class Solution {
    public:
        vector<int> plusOne(vector<int>& digits) {
        int n = digits.size();
        digits[n-1]++;
        if(digits[n-1]<10)
            return digits;
        else
            {
                digits[n-1]=0;
                int count=1;
                while(digits[n-1-count] == 9 && count < n)  count++;
                if(count<n)
                    {
                        digits[n-1-count]++;
                        for(int i = n-1;i>n-1-count;i--)
                            digits[i]=0;
                        return digits;
                    }
                else
                    {
                        digits[0]=1;
                        for(int i =1;i<n-1;i++)
                            digits[i]=0;
                        digits.push_back(0);
                        return digits;
                    }
            }


    }   
};

Leetcode刷题记录11-Remove Duplicates from Sorted Array

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

Given a sorted array, remove the duplicates in place such that each element appear only once and return the new length.

Do not allocate extra space for another array, you must do this in place with constant memory.

For example,
Given input array nums = [1,1,2],

Your function should return length = 2, with the first two elements of nums being 1 and 2 respectively. It doesn’t matter what you leave beyond the new length.

Solutions:

Just search for new distinct element and put it into the right possition of the arry.

Code in C++:

class Solution {
    public:
        int removeDuplicates(vector<int>& nums) {
            if(nums.size() == 0) return NULL;
            if(nums.size() == 1) return 1;
            else
                {
                    int count = 1;
                    int temp = nums[0];
                    for(int i = 0;i<nums.size();i++)
                        {
                            if(nums[i]!= temp) 
                                {
                                    temp = nums[i];
                                    nums[count++]= nums[i];
                                } 
                        }
                    return count;
                }

            }
    };

Problem Description:

Given an integer, write a function to determine if it is a power of two.

Solutions:

Check recuirsively if n can be completely divided by two.

Code in C++:

class Solution {
    public:
    bool isPowerOfTwo(int n) {
        while(n>1)
            {
                if(n%2>0) return false;
                else n=n/2;
            } return n == 1;

        }
    };

Leetcode刷题记录9-Remove Elements

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

Given an array and a value, remove all instances of that value in place and return the new length.

The order of elements can be changed. It doesn’t matter what you leave beyond the new length.

Solutions:

Compare every element with the givin value, ignore it when matches, move other elements forwards.

Code in C++:

class Solution {
    public:
        int removeElement(vector<int>& nums, int val) {
        if(nums.size() == 0) return 0;
        int count1 = 0;
        for(int i =0;i<nums.size();i++)
            {
                if(nums[i]!=val) nums[count1++]=nums[i];
            }
        return count1;

    }
};

Leetcode刷题记录8-Reverse Linked List

发表于 2015-10-06   |   分类于 Leetcode   |  

Problem Description:

Reverse a singly linked list.

Solutions:

For every single linknode from the second link, just put it after the first linknode. Then put the previous head to the bottom of the linked list.(Make a circle and then break it at proper possition)

Code in C++:

/**
* Definition for singly-linked list.
* struct ListNode {
*     int val;
*     ListNode *next;
*     ListNode(int x) : val(x), next(NULL) {}
* };
*/
class Solution {
    public:
        ListNode* reverseList(ListNode* head) {
        if(head == NULL) return NULL;
        if(head->next == NULL) return head;
        ListNode* p;
        ListNode* q;
        p = head->next;
        while(p->next!=NULL)
            {
                q=p->next;
                p->next = q->next;
                q->next = head->next;
                head->next = q;
            }
        p->next = head;
        head = p->next->next;
        p->next->next = NULL;

        return head;

    }
};
