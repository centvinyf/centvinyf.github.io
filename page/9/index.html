Leetcode刷题记录13-Pascal's Triangle

发表于 2015-10-06   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given numRows, generate the first numRows of Pascal’s triangle.

For example, given numRows = 5,
Return

[
[1],
[1,1],
[1,2,1],
[1,3,3,1],
[1,4,6,4,1]
]

Solutions:

For every element after the second line, we can see that a[i][j]=a[i-1][j-1]+a[i-1][j], finally put a 1 at the back of the row.

Code in C++:

class Solution {
    public:
        vector<vector<int>> generate(int numRows) {
        vector<vector<int>> a;
        if(numRows == 0) return a;
        vector<int> last = {1};
        a.push_back(last);
        if(numRows == 1) 
            return a;
        last.push_back(1);
        a.push_back(last);
        if(numRows == 2) 
            return a;
        if(numRows>=3)
            {
                for(int i =2;i< numRows;i++)
                    {
                        last.clear();
                        last.push_back(1);
                        for(int j = 1; j < i; j++)
                            last.push_back(a[i-1][j-1]+a[i-1][j]);
                        last.push_back(1);
                        a.push_back(last);
                    }
                return a;   
            }


        }
    };

Leetcode刷题记录12-Plus One

发表于 2015-10-06   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a non-negative number represented as an array of digits, plus one to the number.

The digits are stored such that the most significant digit is at the head of the list.

Solutions:

It’s a easy task, just pay attention to all possible cases.

Code in C++:

class Solution {
    public:
        vector<int> plusOne(vector<int>& digits) {
        int n = digits.size();
        digits[n-1]++;
        if(digits[n-1]<10)
            return digits;
        else
            {
                digits[n-1]=0;
                int count=1;
                while(digits[n-1-count] == 9 && count < n)  count++;
                if(count<n)
                    {
                        digits[n-1-count]++;
                        for(int i = n-1;i>n-1-count;i--)
                            digits[i]=0;
                        return digits;
                    }
                else
                    {
                        digits[0]=1;
                        for(int i =1;i<n-1;i++)
                            digits[i]=0;
                        digits.push_back(0);
                        return digits;
                    }
            }


    }   
};

Leetcode刷题记录11-Remove Duplicates from Sorted Array

发表于 2015-10-06   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a sorted array, remove the duplicates in place such that each element appear only once and return the new length.

Do not allocate extra space for another array, you must do this in place with constant memory.

For example,
Given input array nums = [1,1,2],

Your function should return length = 2, with the first two elements of nums being 1 and 2 respectively. It doesn’t matter what you leave beyond the new length.

Solutions:

Just search for new distinct element and put it into the right possition of the arry.

Code in C++:

class Solution {
    public:
        int removeDuplicates(vector<int>& nums) {
            if(nums.size() == 0) return NULL;
            if(nums.size() == 1) return 1;
            else
                {
                    int count = 1;
                    int temp = nums[0];
                    for(int i = 0;i<nums.size();i++)
                        {
                            if(nums[i]!= temp) 
                                {
                                    temp = nums[i];
                                    nums[count++]= nums[i];
                                } 
                        }
                    return count;
                }

            }
    };

Leetcode刷题记录10-Power Of Two

Problem Description:

Given an integer, write a function to determine if it is a power of two.

Solutions:

Check recuirsively if n can be completely divided by two.

Code in C++:

class Solution {
    public:
    bool isPowerOfTwo(int n) {
        while(n>1)
            {
                if(n%2>0) return false;
                else n=n/2;
            } return n == 1;

        }
    };

Leetcode刷题记录9-Remove Elements

发表于 2015-10-06   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an array and a value, remove all instances of that value in place and return the new length.

The order of elements can be changed. It doesn’t matter what you leave beyond the new length.

Solutions:

Compare every element with the givin value, ignore it when matches, move other elements forwards.

Code in C++:

class Solution {
    public:
        int removeElement(vector<int>& nums, int val) {
        if(nums.size() == 0) return 0;
        int count1 = 0;
        for(int i =0;i<nums.size();i++)
            {
                if(nums[i]!=val) nums[count1++]=nums[i];
            }
        return count1;

    }
};

Leetcode刷题记录8-Reverse Linked List

发表于 2015-10-06   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Reverse a singly linked list.

Solutions:

For every single linknode from the second link, just put it after the first linknode. Then put the previous head to the bottom of the linked list.(Make a circle and then break it at proper possition)

Code in C++:

/**
* Definition for singly-linked list.
* struct ListNode {
*     int val;
*     ListNode *next;
*     ListNode(int x) : val(x), next(NULL) {}
* };
*/
class Solution {
    public:
        ListNode* reverseList(ListNode* head) {
        if(head == NULL) return NULL;
        if(head->next == NULL) return head;
        ListNode* p;
        ListNode* q;
        p = head->next;
        while(p->next!=NULL)
            {
                q=p->next;
                p->next = q->next;
                q->next = head->next;
                head->next = q;
            }
        p->next = head;
        head = p->next->next;
        p->next->next = NULL;

        return head;

    }
};

Leetcode刷题记录7-Roman to Integer

发表于 2015-10-06   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given a roman numeral, convert it to an integer.

Input is guaranteed to be within the range from 1 to 3999.

Solutions:

Because the range of number is within 3999, so we don’t need to give 1000 times to a character. See the Roman numerals for more infomation.

Code in C++:

class Solution {
    public:

        int RomanToNumbers(char x)
            {
                switch(x)
                    {
                        case 'I': return 1;
                        case 'V': return 5;
                        case 'X': return 10;
                        case 'L': return 50;  
                        case 'C': return 100;  
                        case 'D': return 500;  
                        case 'M': return 1000;  
                        default: return 0;  
                    }
            }
        int romanToInt(string s) {
                if(s.size()==0) return 0;
                if(s.size()==1) return RomanToNumbers(s[0]);
                else
                {
                        int result = RomanToNumbers(s[0]);
                        for(int i =1; i <s.size();i++)
                            {
                                if(RomanToNumbers(s[i])>RomanToNumbers(s[i-1]))
                                result+= (RomanToNumbers(s[i])-2*(RomanToNumbers(s[i-1])));
                                else
                                    {
                                        result+=RomanToNumbers(s[i]);
                                    }
                            }
                        return result;
                }

            }
    };

Leetcode刷题记录6-valid anagram

发表于 2015-10-03   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given two strings s and t, write a function to determine if t is an anagram of s.

For example,
s = “anagram”, t = “nagaram”, return true.
s = “rat”, t = “car”, return false.

Note:
You may assume the string contains only lowercase alphabets.

Solutions:

Because there are only lowercase alphabets, we can simply use an array to hash the strings to keep the number of every character.

Code in C++:

class Solution {
    public:
        bool isAnagram(string s, string t) {
            if(s.size()==0 && t.size()==0) return true;
            if(s.size()!=t.size()) return false;
            int count[26]={0};
            for(int i =0;i<s.size();i++)
                {
                    ++count[s[i]-'a'];
                }
            for(int i =0;i<t.size();i++)
                {
                    if(--count[t[i]-'a']<0) return false;
                }
            return true;
        }
    };

Leetcode刷题记录5-Contains Duplicate

发表于 2015-10-03   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Given an array of integers, find if the array contains any duplicates. Your function should return true if any value appears at least twice in the array, and it should return false if every element is distinct.

Solutions:

This problem is primarily about using the method of hash table. But I really don’t know how to build hash table in C++. So I sorted the array to see if any number is equal to its neighbor.

Code in C++:

class Solution {
    public:
        bool containsDuplicate(vector<int>& nums) {
        if(nums.size()>0)
            {
                std::sort(nums.begin(),nums.end());
                if(nums.size()==2)
                {
                        return nums[0]==nums[1];
                }
                else
                {
                    for(int i = 1; i< nums.size()-1;i++)
                        {
                            if(nums[i]== nums[i-1]|| nums[i]==nums[i+1])
                            return true;
                        }
                }
            }
        return false;

    }
};

Leetcode刷题记录4-Number of 1 Bits

发表于 2015-10-03   |   分类于 Leetcode   |     |   阅读次数

Problem Description:

Write a function that takes an unsigned integer and returns the number of ’1’ bits it has (also known as the Hamming weight).

For example, the 32-bit integer ’11’ has binary representation 00000000000000000000000000001011, so the function should return 3.

Solutions:

I didn’t have that brilliant idea, This Article is a perfect explaination.

Code in C++:

class Solution {
    public:
        int hammingWeight(uint32_t n) {
        int count= 0;
        while(n!=0)
            {
                n &= (n-1);
                count++;
            }
        return count;
    }
};
